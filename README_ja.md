## 🚀 インストールガイド

以下の簡単な手順で拡張機能をインストールします。

1. ### ダウンロードと解凍

    .zipファイルをダウンロードし、お使いのコンピュータのフォルダに解凍（展開）します。

2. ### Chrome拡張機能を開く

    Chromeブラウザのアドレスバーに `chrome://extensions` と入力し、拡張機能ページに移動します。

3. ### デベロッパーモードを有効にする
    ![](./assets/dev-mode.png)
4. ### 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックする
    ![](./assets/load-unpacked.png)
5. ### ステップ1で解凍した拡張機能のディレクトリを選択する

## ⚙️ 利用開始

APIベースURLとAPIキーの設定方法
Claude拡張機能のアイコンを右クリックし、「Options」（オプション）メニューをクリックしてオプションページを開きます。

オプションページで「API configuration (internal)」（API設定（内部））をクリックし、Anthropic API Keyの下にある入力欄にAPIキーを入力します。

サードパーティのAPIキーを設定する必要がある場合は、APIベースURLを設定する必要があります。https://cfc.aroic.workers.dev/ にログインし、Base URLの下にある入力欄に設定します。一部のバージョンでは、拡張機能のオプションページでこれを設定することもできます。

これで設定は完了です！ChromeでClaudeを直接お楽しみください。
